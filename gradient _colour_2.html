Spectra Colour Palette Creator

Instructions

In order to use this Colour Palette Maker first, use the colour bar to select the hue(colours like red, green,blue,etc) that you would like to work with

Then use the main window to alter the shade(how dark or light it is). you can see this in the smaller window beside the main one.

once you are happy with your colour you can press the "add colour" button below and add that colour to your palette
